# Introduction

Naninovel is an extension for Unity game engine. It consists of a C# framework and editor utilities to support development of visual novel games.

# Features

The following features are already implemented; more will be added as the engine undergoes an active development:

  • Document-based novel scripts
    • Generic text lines
    • Labels
    • Script defines
    • Actions inlining
  • Text printers
    • Dialogue printer
    • Fullscreen printer
    • TextMesh Pro support
  • Characters
    • Sprite characters
    • Diced sprite characters
    • Animated characters
    • Live2D characters
  • Backgrounds
    • Sprite backgrounds
    • Video backgrounds
    • Animated backgrounds
    • Scene backgrounds
  • Background transition effects
  • Special effects (FX system)
  • Background music (BGM)
  • Sound effects (SFX)
  • Voicing and auto voicing
  • Movies
  • Choices
  • Custom state variables
  • Conditional script flow
  • Save-load system
  • Game settings
  • Cross-platform input
  • Printer backlog
  • Text skipping
  • Auto-advance text
  • UI toggling
  • Adaptive UI layout
  • UI customization
  • Managed text
  • Localization
    • Novel scripts localization
    • Resources localization
  • Community modding
  • Development console
  • Script rewinding and debug
  • Custom novel actions
  • Custom actor implementations
  • Google Drive integration
